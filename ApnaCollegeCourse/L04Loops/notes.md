# Loops
- For Loops
- While Loops
- Do While Loops

All the loops have same similar structure to that of c++

## For loops
```java
for(int i=0;i<a;i++){
    System.out.println(i+" from "+a);
}
```

## While Loop
```java
while(a>0){
    System.out.println("The value of a is "+a);
    a--;
}
```

## Do-While Loop
```java
do{
    System.out.println("the value of a is "+a);
    a--;
}while(a>0);
```
